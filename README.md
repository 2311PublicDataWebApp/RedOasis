## ⚙주요기능 상세 설명




|기능|내용|
|------|---|
|와인 설명|와인에 대한 기초지식을 설명하는 페이지입니다.|
|와인설명 리스트 조회|회원은 와인설명글을 목록으로 조회할 수 있습니다.|
|상세 게시글 조회|목록에서 게시글을 클릭하면 상세 내용을 확인할 수 있습니다.|
|페이징 기능|게시글 리스트에서 9개씩 조회할 수 있습니다. 페이징 기능을 통해 페이지를 이동할 수 있습니다.|


|기능|내용|
|------|---|
|와인 시음회|와인 시음회가 열리는 정보를 보여주는 페이지입니다.|
|와인 시음회 리스트 조회|회원은 와인시음회글을 목록으로 조회할 수 있습니다.|
|상세 게시글 조회|목록에서 게시글을 클릭하면 상세 내용을 확인할 수 있습니다.|
|페이징 기능|게시글 리스트에서 9개씩 조회할 수 있습니다. 페이징 기능을 통해 페이지를 이동할 수 있습니다.|


|기능|내용|
|------|---|
|와인 강의|와인강의를 학습할 수 있는 페이지입니다.|
|와인강의 리스트 조회|회원은 와인강의글을 목록으로 조회할 수 있습니다.|
|상세 게시글 조회|목록에서 게시글을 클릭하면 상세 내용을 확인할 수 있습니다.|
|페이징 기능|게시글 리스트에서 9개씩 조회할 수 있습니다. 페이징 기능을 통해 페이지를 이동할 수 있습니다.|


와인
|기능|내용|
|------|---|
|와인 목록|와인 리스트를 조회할 수 있습니다.|
|카테고리|와인 종류에 따라 리스트를 필터링해서 고칠 수 있습니다.|
|검색|와인 이름을 기준으로 키워드를 쓰면 특정 와인을 검색해서 조회할 수 있습니다.|
|페이징 기능|1페이지당 15개씩 항목의 와인 리스트를 조회할 수 있습니다.|
|와인 상세 조회|와인의 정보를 상세 조회할 수 있습니다.|
|댓글 및 별점|와인 상세 정보 페이지에 댓글을 달 수 있으며 점수를 설정하고 댓글을 달면 점수에 따라 와인의 점수가 변동합니다.|


와이너리
|기능|내용|
|------|---|
|와이너리 지도|등록된 와이너리들의 위치를 확인할 수 있으며 해당 와이너리 위치에 위치한 마커를 클릭 시 와이너리 상세조회 페이지로 이동할 수 있습니다.|
|와이너리 상세 조회|와이너리의 정보를 상세 조회 할 수 있습니다.|
|와이너리 댓글|댓글을 작성 및 삭제할 수 있으며 자신이 작성한 댓글만 삭제할 수 있습니다.|


|기능|내용|
|------|---|
|로그인|  사용자는 등록된 아이디와 비밀번호를 사용하여 로그인할 수 있습니다. 사용자의 로그인 상태는 세션을 통해 유지하고 인증된 요청만을 처리합니다. |
|회원가입|  사용자는 이메일 자신의 개인정보를 입력하여 새로운 계정을 생성할 수 있습니다.|
|회원 삭제| 사용자가 회원 탈퇴를 요청할 경우 해당 계정과 관련된 모든 정보를 안전하게 삭제합니다.|
|아이디 찾기|  등록된 이메일 주소를 통해 사용자의 아이디를 확인할 수 있습니다.|
|비밀번호 찾기|  등록된 아이디와 이메일 주소를 통해 사용자의 비밀번호를 변경할 수 있습니다.|
|아이디 저장|  사용자가 웹 브라우저를 닫았다가 다시 열어도 최근 로그인한 계정의 아이디가 미리 입력될 수 있게 설정해놨습니다.|


|기능|내용|
|------|---|
|자유게시판 목록| 사용자는 사람들들이 올린 자유게시판을 볼 수 있습니다. 수정과 삭제를 할 수 있습니다.|
|자유게시판 등록| 사용자는 자유게시판 제목, 내용, 첨부파일을 등록할 수 있습니다.|
|자유게시판 수정| 사용자는 자유게시판 제목, 내용, 첨부파일을 수정할 수 있습니다.|
|자유게시판 삭제| 사용자는 자유게시판을 삭제 할 수 있습니다.|


|기능|내용|
|------|---|
|와인 등록|와인 정보를 등록해 와인 목록에 노출시키거나 메인화면에 이달의 와인으로 등록할 수 있습니다.|
|와인 조회|모든 와인 정보를 조회할 수 있습니다.|
|와인 수정|특정 와인 정보를 수정할 수 있습니다. (와인 번호 제외)|
|와인 삭제|특정 와인을 삭제할 수 있습니다.|
|부가 기능|특정 항목기준 정렬, 페이지당 컬럼 갯수, 검색, 조건에 맞는 페이징|


|기능|내용|
|------|---|
|와이너리 등록|와이너리 정보를 등록해 지도에 해당 위치를 노출시킬 수 있습니다.|
|와이너리 조회|모든 와이너리 정보를 조회할 수 있습니다.|
|와이너리 수정|특정 와이너리 정보를 수정할 수 있습니다. (아이디 제외)|
|와이너리 삭제|특정 와이너리을 삭제할 수 있습니다.|
